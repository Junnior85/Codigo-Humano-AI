# Codigo-Humano-AI
Descripción de Codigo Humano AI: Un compañero de IA ("Living Diary") diseñado para ofrecer apoyo emocional, validación y detección temprana de patrones de riesgo comórbidos (Ansiedad + Depresión) mediante análisis longitudinal de conversaciones. Basado en Llama 3 (Groq) y Streamlit.
